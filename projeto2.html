📌 Sobre o Projeto

Este projeto consiste em um RPA (Robotic Process Automation) desenvolvido para automatizar a escrituração de notas fiscais no sistema SAP. O RPA substituiu tarefas manuais, garantindo maior eficiência e precisão nas escrituração fiscais diárias.

🚀 Objetivo do Projeto

  • Automatizar o processo de escrituração de notas fiscais no SAP.
  • Reduzir o tempo gasto manualmente pelo colaborador.
  • Evitar erros manuais na escrituração fiscal.

🧠 Tecnologias Utilizadas

  • Python para automação e processamento de dados.
  • Pandas para manipulação e organização de planilhas.
  • PyAutoGUI para automação de ações no SAP.

🔍 Funcionalidades

  • Leitura de planilhas de notas fiscais.
  • Automação das entradas no sistema SAP.
  • Validação de dados para evitar inconsistências.
  • Geração de logs de execução para acompanhamento.

📊 Resultados / Benefícios

  • Redução de mais de 4 horas diárias de trabalho manual.
  • Maior precisão e confiabilidade na escrituração fiscal.
  • Otimização do tempo do colaborador para tarefas estratégicas.
